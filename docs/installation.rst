============
Installation
============


Stable release
--------------

To install Python UDM REST Client, run this command in your terminal:

.. code-block:: console

    $ pip install udm_rest_client

This is the preferred method to install Python UDM REST Client, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Python UDM REST Client can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/univention/python-udm-rest-api-client

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/univention/python-udm-rest-api-client/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python3 setup.py install


.. _Github repo: https://github.com/univention/python-udm-rest-api-client
.. _tarball: https://github.com/univention/python-udm-rest-api-client/tarball/master
